# DataFlow Pro

DataFlow Pro is an advanced database management system built with pure HTML, CSS, and JavaScript, using Supabase as the backend. This project provides comprehensive features for data management, viewing statistics and analytics, and executing SQL queries.

## ‚ú® Features

-   **Full Record Management (CRUD)**: Add, view, edit, and delete records.
-   **Supabase Integration**: Uses a real Postgres database for stability and scalability.
-   **File Uploads**: Upload and display profile pictures using Supabase Storage.
-   **Advanced Search**: Fast, server-side searching.
-   **Analytics Dashboard**: Displays overall data statistics and analytical charts.
-   **Data Export**: Ability to export data in PDF, CSV, and JSON formats.
-   **Modern UI**: Beautiful and responsive design.

## üöÄ Getting Started

1.  Clone the project:
    ```bash
    git clone https://github.com/your-username/DataFlow-Pro.git
    ```
2.  Navigate to the project directory:
    ```bash
    cd DataFlow-Pro
    ```
3.  Create a new project on Supabase and run the SQL script provided in the documentation to create the `users` table.
4.  In the `js/api.js` file, replace the `SUPABASE_URL` and `SUPABASE_ANON_KEY` values with your Supabase project credentials.
5.  Open the `index.html` file in your browser.
